## Developer

### How to get started?

__Prerequisites:__ 

*   Node/NPM
*   Hugo


#### Installation

```sh
$   npm install
```


#### Development

```sh
$   npm start
```


#### Build and Deploy

```sh
$   npm build[:prod]
```
